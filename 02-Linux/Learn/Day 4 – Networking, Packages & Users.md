
### Networking Basics:
* How linux talks to the internet
* Linux connects to the internet using a network interface
* ping google.com
	* Send packets to google and wait for reply
* curl website
	* Fetches data from a website or API using the command
* wget
	*   Download files from internet
* netstat
	* Shows network connection, ports and service running

### Package Manager:
* Installing software in OS
* `sudo apt update`
	* Update software list
* `apt install tree`
	* Install program called tree
* `apt upgrade`
	* Install latest version of all installed packages

### User Management:
* Very important to maintain user in OS
* `sudo useradd testuser`
	* Create new user named test user
* `sudo passwd testuser`
	* Set or change the password of user `testuser`
