
### DevOps work overview:
* Let consider 3 developer and 1 devops person in a team
* DevOps responsibility are
	* Create/enforcing branching model
	* Review and Merge PR's
	* Resolve conflicts
	* Manage releases
	* Handle hotfix
	* Rollback
	* Clean history
	* Apply rebase/squash
	* Use cherry pick
	* Recover from mistakes using reflog
	* Trigger CI with tags
